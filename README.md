# Double Pendulum Video
This is the script used to generate the visualisation in https://youtu.be/n7JK4Ht8k8M

It will create ~200MiB of data while rendering and takes a few hours.

## Requirements
The following Python libraries are required:
 * Manim
 * pillow

Run this command to install them automatically:

`pip install -r requirements.txt`

The Manim library also requires the presence of a LaTeX compiler with standalone.cls.
This can be installed on Ubuntu with this apt command:

`sudo apt-get install texlive-latex-extra`
